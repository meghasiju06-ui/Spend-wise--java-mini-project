 Problem Statement
Managing daily expenses manually is inefficient and prone to errors. A digital system is required to record, update, and track expenses in an organized manner.
 Objective:
To develop a Java-based desktop application that allows users to:
Login securely
Add expenses
Edit expenses
Delete expenses
View all expenses in tabular format
 Features:
User Login Authentication
Add Expense
Edit Expense
Delete Expense
Display Data in JTable
Database Connectivity using JDBC
Exception Handling
2-Tier Architecture
Technologies Used:
Java
Java Swing
JDBC
MySQL
